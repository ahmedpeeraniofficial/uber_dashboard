# uber_dashboard
Uber Analytics Dashboard | Power BI  Interactive Power BI dashboard analyzing Uber bookings, revenue, vehicle performance, and ratings using key KPIs like total bookings, revenue, and distance. Built with Power BI, DAX, and data modeling. 🚀

🚖 Uber Analytics Dashboard | Power BI
📌 Project Overview

This project showcases an interactive Uber Analytics Dashboard built in Power BI, designed to transform raw ride-booking data into meaningful and business-ready insights. The dashboard helps analyze overall performance, revenue trends, operational efficiency, and customer experience across multiple vehicle categories.

📊 Key KPIs
Completed Bookings
Lost Bookings
Total Revenue
Total Distance Covered
Average Trip Distance

🚘 Vehicle Categories Analyzed
Auto
Bike
Go Mini
Go Sedan
Premier Sedan
Uber XL

📅 Time-Based Analysis
Monthly Booking & Revenue Trends
Quarterly Booking & Revenue Trends

📊 Key Insights
Revenue analysis by vehicle type
Top pickup and drop-off locations based on booking volume
Vehicle-wise performance comparison using interactive filters
Customer & driver experience tracking through average ratings

💡 Business Value

This dashboard provides a complete 360° view of Uber operations, helping stakeholders understand demand patterns, revenue performance, location trends, and service quality for better decision-making.

Power BI | DAX | Data Modeling | Data Visualization | Business Intelligence
